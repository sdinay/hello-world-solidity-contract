Installation:
$ npm i -g solc
$ npm i -g truffle
$ npm i -g ganache-cli

$ mkdir hello-world-solidity-contract
$ cd hello-world-solidity-contract

$ truffle init
$ truffle compile

$ ganache-cli -p 7545
$ truffle migrate
$ truffle test ./test/hello_eth_salon.js
